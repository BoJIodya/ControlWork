Задача:
Написать программу, которая из имеющегося массива строк формирует массив из строк,
длина которых меньше либо равна 3 символа. Первоначальный массив можно ввести с клавиатуры,
либо задать на старте выполнения алгоритма. При решение не рекомендуется
пользоваться коллекциями, лучше обойтись исключительно массивами.

Алгоритм решения:

Объявляем два массива, первый- array string, исходный, втрой array result- в него запишем результат вычислений.

Вычисления производим в методе. Где проходим каждый элемент массива, считывая его (элемента) размер.
Если размер удовлетворяет условию (<= 3), то записываем этот елемент в массив array result.

При этом, что бы не было пустых элементов в конечном массиве, вводим счетчик положительных (удовлетворяющих условию) результатов.

При присвоении массиву array result нового члена, к счетчику прибавляем 1.
Цикл метода заканчивается при достижении конца массива array string.
На выход передается новый массив array result, его выводим на печать.
